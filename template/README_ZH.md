## 项目简介 <!-- 必填 -->

本项目是一款面向数据处理与文本挖掘方向的智能助手系统 。系统依托于局域网集群部署的大语言模型，旨在对海量非结构化的中英文文本（如课程资料、学术论文、作业报告等）进行深度的结构化提取与多维分析 。核心功能包括智能摘要、情感分析以及关键特征提取，进阶功能支持批量处理流水线和多文档多维对比分析 。

## 技术亮点

* **提示词工程演进 (Prompt Engineering):** 引入 System Role、Few-shot (少样本提示) 和思维链 (CoT) 推理，确保输出格式极其稳定，并攻克了复杂的逻辑推理任务。
* **超长上下文管理 (Context Window Management):** 实现级联摘要生成和自适应上下文分配算法，完美解决长文本导致的 OOM (显存溢出) 异常。
* **API 生命周期管理:** 设计了具备线性退避重试策略的高鲁棒性错误处理机制，有效缓解服务器限流压力并避免“重试风暴” 。
* **异构数据解析与鲁棒性:** 实现了多态数据流的归一化处理，并加入基于观察者模式的前置熔断机制，可自动隔离损坏文件而不导致整个流水线崩溃。
* **异步并发调度:** 使用 `ThreadPoolExecutor` 并行化执行子任务，有效掩蔽网络 I/O 延迟，大幅提升流水线吞吐量。
* **I/O 架构优化与前端:** 采用双模态 I/O (含服务器端本地直读) 突破 Jupyter WebSocket 传输瓶颈，并使用原生 `ipywidgets` 构建了内嵌式 GUI 交互界面。


## 结果 / 演示

* 实时文件仪表盘: 提供文件解析状态、提取字符数及预估 Token 消耗的实时监控，包含针对长文本超限的智能提醒。
* 批量分析流水线: 突破单次请求限制，自动格式化输出包含摘要、情感打分及关键词的统一 Markdown 报告，实现机器级别的高效审阅。
* 多文档对比报告: 打破单一文本限制，系统可自动汇总并生成包含“核心主旨、共同点、核心差异维度”的结构化深度分析报告 。
